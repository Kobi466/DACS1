🍽️ Ý TƯỞNG ỨNG DỤNG: "SMART RESTAURANT GUI"
👤 1. GUI KHÁCH HÀNG – "Customer App"
Giao diện cho người dùng đặt món, đặt bàn, nhắn tin.

Chức năng:
Mô tả	Bảng liên quan
Đăng nhập / đăng ký	Customer
Đặt món ăn	Order, OrderItem, MenuItem
Đặt bàn trước	Reservation, TableBooking
Chat với nhà hàng	Message
Xem đánh giá và gửi đánh giá	Rating
Áp dụng mã giảm giá	CustomerVoucher, Voucher
Mô phỏng màn hình:
🧾 Danh sách món ăn (menu)

🪑 Danh sách bàn/phòng trống

💬 Chat box

🧧 Nhập mã giảm giá

✅ Nút “Gửi đơn hàng”

👨‍🍳 2. GUI NHÀ HÀNG – "Staff Dashboard"
Dành cho nhân viên quán để xử lý đơn và tương tác.

Chức năng:
Mô tả	Bảng liên quan
Nhận đơn hàng mới	Order, OrderItem
Chuyển món vào bếp	KitchenQueue
Trạng thái bếp	KitchenQueue
Quản lý đặt bàn	Reservation, TableBooking
Phản hồi đánh giá	Rating, Message
Gửi phản hồi chat	Message
Mô phỏng màn hình:
📦 Đơn hàng mới (với món ăn chi tiết)

🔥 Hàng đợi bếp: đang nấu / xong

🪑 Đặt bàn chờ xác nhận

💬 Chat box phản hồi khách

🔄 Giao tiếp 2 GUI thông qua Database
Sử dụng bảng Message, Order, Reservation, KitchenQueue để truyền tải dữ liệu giữa hai GUI.

Cập nhật qua polling hoặc thread mỗi 3s

Khi khách nhấn "Gửi", Message và Order được lưu

GUI nhà hàng sẽ đọc tin mới từ DB và phản hồi🧩 CÁC BẢNG (ENTITY) ĐÃ CÓ
                                             Entity	Mô tả
                                             Customer	Thông tin người dùng
                                             Order	Đơn hàng từ khách
                                             OrderItem	Món cụ thể trong đơn
                                             MenuItem	Danh sách món ăn
                                             Reservation	Đặt bàn
                                             TableBooking	Quản lý bàn/phòng
                                             Message	Tin nhắn giữa khách ↔ nhà hàng
                                             Voucher	Mã giảm giá
                                             CustomerVoucher	Voucher được áp dụng
                                             KitchenQueue	Hàng đợi món bếp đang xử lý
                                             Rating	Đánh giá khách hàng


Thành phần	Mô tả vai trò chính	Công nghệ sử dụng
💬 Chat trung gian (ChatServer)	Trung gian kết nối các client, nhận/gửi tin nhắn	Java Socket Server (TCP)
👤 GUI KHÁCH HÀNG (CustomerGUI)	Gửi/nhận tin nhắn từ server	Java Swing Client
👨‍🍳 GUI NHÀ HÀNG (StaffGUI)	Nhận/gửi tin nhắn từ khách hàng	Java Swing Client
🗃️ Lưu lịch sử tin nhắn	Ghi lại toàn bộ nội dung chat giữa 2 bên	Hibernate ORM + MySQL
📦 Dữ liệu truyền qua mạng	Định dạng tin nhắn khi gửi/nhận giữa client ↔ server	JSON qua Socket (String)
